总体流程

我们首先执行命令apktool d xxx.apk，然后看输出如下

I: Using Apktool 2.3.1 on douyin.apk
I: Loading resource table...

  • 加载resource table

  • 解码一些资源文件
  • 解码dex文件
  • copy剩余文件
